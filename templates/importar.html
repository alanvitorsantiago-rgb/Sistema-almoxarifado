Importar Itens via Planilha Excel

Instruções

A planilha deve ser um arquivo .xlsx e conter as seguintes colunas na primeira linha:

CÓDIGO, CÓDIGO OPCIONAL, TIPO, LOCAL, DESCRIÇÃO, UN., DIMENSÃO, CLIENTE, LOTE, ITEM NF, NF, VALIDADE, ESTAÇÃO, QTD ESTOQUE

  • A coluna CÓD.ESTOC. (ID interno) não deve ser incluída na planilha.
  • As colunas CÓDIGO, DESCRIÇÃO, LOTE, ITEM NF, QTD ESTOQUE são obrigatórias para cada linha.
  • A data de validade deve estar no formato AAAA-MM-DD (ex: 2025-12-31).
  • O sistema irá criar novos itens se o código não existir. Se o código já existir, ele adicionará um novo lote (ou somará a quantidade se o lote/item_nf já existir para aquele item).
